# 🤖 AI Smart Photo Analyzer

An intelligent image analysis web app built with **Streamlit** 
and **Azure Computer Vision API**.

## Features
- ✦ AI-generated image captions with confidence score
- 😀 Face detection with age and gender estimation  
- # Scene tags with confidence percentages
- ⬡ Object detection with confidence bars
- ◐ Color palette extraction

## Tech Stack
- Python
- Streamlit
- Azure Cognitive Services (Computer Vision v3.2)
- Pillow (PIL)

## How to Run
1. Clone this repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add your Azure API key and endpoint in app.py
4. Run the app:
   streamlit run app.py
